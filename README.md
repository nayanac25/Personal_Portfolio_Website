# Frontend Developer Portfolio

A modern and responsive personal portfolio website built using **HTML, CSS, and JavaScript** to showcase my projects, skills, and frontend development journey.

- Features
- Responsive design for desktop, tablet, and mobile screens
- Light / Dark theme toggle
- Sticky navigation bar with active section highlight
- Typing animation in the hero section
- Scroll reveal animations
- Project showcase section with Live Demo / GitHub links
- Functional contact form
- Download CV button
- Clean and modern UI design

- Tech Stack
- HTML5
- CSS3
- JavaScript
- Unicons
- ScrollReveal.js
- Typed.js
- Formspree (for contact form handling)

- Sections Included
- Home
- About
- Skills
- Projects
- Contact
- Footer

- Project Overview
This portfolio website was created to present my profile, technical skills, and projects in a professional way for recruiters and interviewers.
It reflects my interest in building responsive, user-friendly, and visually appealing web interfaces.

-Key Highlights
- Built a fully responsive single-page portfolio website
- Implemented dark/light theme toggle 
- Added scroll-based active navigation highlighting
- Integrated typing animation and scroll reveal effects
- Added project cards with Live Demo and GitHub links
- Made the contact form functional using Formspree
